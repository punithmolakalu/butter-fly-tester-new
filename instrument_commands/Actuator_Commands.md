# Actuator Commands

This document describes the commands used for controlling actuators.

## Commands

### HOME BOTH
Homes both actuators simultaneously.

**Usage:**
```
HOME BOTH
```

### homea
Homes actuator A.

**Usage:**
```
homea
```

### homeb
Homes actuator B.

**Usage:**
```
homeb
```

### movea <distance>
Moves actuator A by the specified distance.

**Usage:**
```
movea <distance>
```

**Parameters:**
- `<distance>` - The distance to move actuator A (units may vary based on system configuration)

### moveb <distance>
Moves actuator B by the specified distance.

**Usage:**
```
moveb <distance>
```

**Parameters:**
- `<distance>` - The distance to move actuator B (units may vary based on system configuration)
